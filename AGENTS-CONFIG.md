# 🤖 多 Agent 配置

> 快速响应 vs 深度思考，按需切换

---

## Agent 清单

| Agent | 模型 | 用途 | 特点 |
|-------|------|------|------|
| **agent-quick** | glm-4.5-air | 简单问题 | ⚡ 快速响应 |
| **agent-deep** | glm-5 | 复杂问题 | 🧠 深度思考 |

---

## 使用方式

### 简单问题 → agent-quick
```
/quick 你的问题
```
- 改文字、加按钮
- 查询信息
- 简单配置

### 复杂问题 → agent-deep
```
/deep 你的问题
```
- 架构设计
- 复杂功能开发
- 问题排查
